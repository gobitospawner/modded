# Animal Company Mod Menu Spawner

## Description
The Animal Company Mod Menu Spawner is a versatile tool for gamers who want to enhance their gameplay experience by enabling various mod options seamlessly.

## Features
- **User-Friendly Interface**: Easy to navigate menu for selecting mods.
- **Customizable Options**: Tailor your mods to fit your playstyle.
- **Compatibility**: Works with various game versions.

## Installation
1. Download the mod menu files from the repository.
2. Place the files in your game’s mod directory.
3. Launch the game and access the mod menu from the main interface.

## Usage
- Open the mod menu using the designated keybind.
- Navigate through the options and toggle the mods you wish to activate.
